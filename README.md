## 📘 What this is

awesome-automated-research is a curated list of autonomous research systems and tools. It helps you find software, models, and platforms that can collect, sort, and review information with less manual work.

Use this list if you want to:
- compare research tools in one place
- find systems that support web search, note taking, and source tracking
- review tools for personal, academic, or team use
- save time when choosing a research workflow

## 💻 What you need

Before you start, make sure your Windows PC can:
- open a web browser
- download files
- extract ZIP files if the release comes in a ZIP package
- open a text file or README file

A modern Windows 10 or Windows 11 system works well for this kind of content browser. If your browser is current, you should be fine.

## 🔍 How to use it

Start with the category that matches your goal.

If you want to:
- gather sources faster, look at search and crawl tools
- summarize long text, look at paper and document tools
- manage many links, look at source tracking tools
- build a research workflow, look at agent and orchestration tools

A good way to use the list is to:
1. scan the category names
2. open one tool at a time
3. check what each tool does
4. compare setup effort, use case, and output format
5. keep the ones that fit your work

## ⚙️ Simple troubleshooting

### The file will not open
Try these steps:
- check that the download finished
- right-click the file and open it again
- if it is a ZIP file, extract it first
- try a different browser if the download fails

### The page shows many files
Download the newest release file at the top. If you are not sure, choose the file with the latest version name or the most recent date.

### Windows asks for permission
If Windows shows a security prompt, check the file name and source first. Make sure you downloaded it from the release page above.

### The text looks broken
Try opening the file in:
- Microsoft Edge
- Google Chrome
- Notepad
- a PDF reader

## 🧭 Picking the right tool

Not every research tool works the same way. When you compare options from the list, look for:
- clear output
- source links
- support for long documents
- easy export options
- simple setup
- Windows-friendly use

If you want a tool that fits daily use, pick one that keeps the workflow simple and shows where its answers come from.
